Project Name

new KT.com 리뉴얼

Project Period

2017년 2월 ~ 2017년 10월(9개월)

Role & Result

KT.com 리뉴얼 을 위한 퍼블리싱 지원

주로 서브 상세페이지들을 작업했고 일부는 서부 메인 작업도 했습니다. 사파리 포함 크로스 브라우징을 해야 했으며, 웹 접근성 마크를 얻기 위해 웹 접근성도 고려했습니다. 생각보다 작업량이 많아서 각 카테고리별 작업을 진행하며 먼저 완료한 순으로 남은 작업을 지원했습니다. 휴일도 반납하고 마무리 작업에 신경을 많이 썼는데 정성을 들인 것만큼 2017 WEB AWARD KOREA 에서 생활브랜드부문 통합대상을 받아 뜻깊은 프로젝트였습니다.
